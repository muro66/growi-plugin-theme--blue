# スクリプトプラグインで追加できそうな機能（案）

GROWI のクライアント側を拡張するスクリプトプラグインで実現しやすい機能のアイデア一覧です。

---

## 1. ナビゲーション・目次

| 機能 | 説明 | 難易度 |
|------|------|--------|
| **サイドに固定の目次** | 見出し（h2/h3）を抽出して右サイドに固定表示。長文で「今どこか」が分かりやすくなる | 中 |
| **「一番上へ」ボタン** | スクロールで表示し、クリックでページトップへ | 低 |
| **前後のページリンク** | 同じ親配下のページを「前のページ」「次のページ」で遷移 | 中 |
| **パンくずの折りたたみ** | 長いパスを省略表示し、ホバーで全文表示 | 低 |

---

## 2. 表示・読みやすさ

| 機能 | 説明 | 難易度 |
|------|------|--------|
| **行番号付きコードブロック** | コードブロックの左に行番号を表示（既存 copy-code と組み合わせ可） | 中 |
| **画像のライトボックス** | 画像クリックで拡大・オーバーレイ表示 | 低〜中 |
| **テーブルの横スクロール固定** | 幅の広いテーブルで見出し列を固定して横スクロール | 中 |
| **フォントサイズ切り替え** | 本文のフォントサイズを S/M/L で切り替え（localStorage に保存） | 低 |
| **印刷用スタイル** | 印刷時にサイドバー非表示・余白調整など | 低 |

---

## 3. 編集・執筆支援

| 機能 | 説明 | 難易度 |
|------|------|--------|
| **編集中の自動保存（下書き）** | 一定間隔で localStorage や API に下書き保存し、クラッシュ時に復元 | 中 |
| **Markdown ショートカット** | 選択範囲を `**` で囲むなど、よく使う装飾をキーボードで実行 | 中 |
| **テンプレート挿入ボタン** | よく使う見出し・テーブル・コードブロックをワンクリックで挿入 | 低 |
| **文字数カウント** | 編集中に文字数・段落数などをリアルタイム表示 | 低 |

---

## 4. 検索・一覧

| 機能 | 説明 | 難易度 |
|------|------|--------|
| **ページ内検索ハイライト** | Ctrl+F の代わりに、プラグインの検索ボックスでページ内の語句をハイライト | 中 |
| **最近開いたページ一覧** | サイドバーやヘッダーに「最近見たページ」を数件表示 | 中 |
| **お気に入り / ピン留め** | よく使うページをピン留めしてサイドバー上部に表示 | 中 |
| **ツリーの折りたたみ記憶** | サイドバーのツリーの開閉状態を localStorage に記憶 | 低〜中 |

---

## 5. 外部連携・便利機能

| 機能 | 説明 | 難易度 |
|------|------|--------|
| **外部リンクを新しいタブで開く** | 外部 URL を自動で `target="_blank"` に（既存設定があれば要確認） | 低 |
| **共有リンクのワンクリックコピー** | 「このページのリンクをコピー」ボタン | 低 |
| **QR コード表示** | ページ URL の QR コードをモーダルで表示（会議・現場で共有用） | 低 |
| **更新日時の相対表示** | 「3日前に更新」のように相対表記を追加 | 低 |

---

## 6. テーマ連携（fm26-blue と合わせる）

| 機能 | 説明 | 難易度 |
|------|------|--------|
| **カスタムボタンのスタイル統一** | プラグインで追加するボタンをテーマの青系・発光縁に合わせる | 低 |
| **目次パネルの縁取り** | 固定目次をテーマのカード風（角丸・青い縁）で表示 | 低 |

---

## まず試すなら（難易度・効果のバランス）

1. **「一番上へ」ボタン** … 実装が軽く、体感の改善が大きい  
2. **共有リンクのコピーボタン** … 需要が多く、実装が簡単  
3. **画像ライトボックス** … よくある要望で、既存プラグインやライブラリの参考が多い  
4. **フォントサイズ切り替え** … アクセシビリティ向上になり、状態管理の練習にもなる  

---

## 実装前に確認したいこと

- **growiFacade** で何が使えるか（markdownRenderer、ルート情報、API など）
- 既存プラグインとの重複（[GROWI プラグイン一覧](https://growi.org/plugins) で「script」タイプを確認）
- サーバー側の変更が必要か（スクリプトプラグインはクライアントのみで完結するか）
