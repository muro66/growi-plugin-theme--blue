# growi-plugin-bookshelf-map

GROWI のページツリーを「本棚ビュー」と「Knowledge Graph（グラフ）」で表示し、自サイトリンクにマウスオーバーでプレビューを表示するスクリプトプラグインです。サイドバーに「📚 本棚」ボタンが追加され、クリックでパネルが開きます。

## 機能

- **本棚ビュー**: 現在ページのパス（またはルート）を「棚」として、その直下の子ページをカード表示。戻る・ルートで表示を切り替え。カードクリックでページへ遷移（子を持つページはその棚を開く）。
- **グラフタブ**: パネル内の「🔗 グラフ」タブで、現在ページを中心に、本文から抽出したリンク先を 2D グラフで表示。ノードクリックで該当ページを開く。
- **リンクプレビュー**: ページ内の自サイトリンク（パス形式のリンク）にマウスオーバーすると、約 0.5 秒後にリンク先のタイトル・要約をポップオーバーで表示。

## 必要な環境

- GROWI（オンプレ / GROWI.cloud）
- ブラウザから `/_api/v3/pages/list` および `/_api/v3/page` が同一オリジンで利用できること

## インストール

1. このリポジトリを GROWI のプラグインとしてインストール（管理画面の「プラグイン」でリポジトリ URL を指定）
2. プラグインを有効化
3. サイドバーに「📚 本棚」が表示されるのでクリックしてパネルを開く

## 開発

```bash
npm install
npm run build
```
